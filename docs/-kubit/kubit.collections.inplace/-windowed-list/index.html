WindowedList

class WindowedList<T>(sourceList: List<T>, windowSize: Int, stepSize: Int = 1, partialWindows: Boolean = false) : AbstractList<InPlaceSubList<T>> (source)

Represents a list of windows as a list of InPlaceSubList created from a source list. Each window is a view into the original list. Changes in the original list will be reflected in the windows.

Parameters

sourceList

The original list.

windowSize

The desired size of each window. Must be greater than 0.

stepSize

The step size between consecutive windows. Defaults to 1.

partialWindows

Whether or not to include partial windows at the end of the list if any. Defaults to false.

Throws

IllegalArgumentException

if windowSize or stepSize is less than or equal to 0
